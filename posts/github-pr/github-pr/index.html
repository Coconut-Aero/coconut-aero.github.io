Linux 笑传之 Windows.h

2024年12月7日2 分钟阅读 闲聊

关于linux仓库被推送带有windows.h的后门那些事

前些天看到了Create usr.c by axhuoga1 · Pull Request #1086 · torvalds/linux - WaybackMachine，被Windows.h给看乐了，后面又看到别人的评论，居然是从某些地方抄来的"后门"，令人忍俊不禁。
